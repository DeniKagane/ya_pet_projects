# Исследование надежности заемщиков


## Данные

В наличии была статистика о платёжеспособности клиентов:
- Количество детей
- Срок работы
- Образование
- Семейный статус
- Рабочий статус
- Пол
- Цель кредита
- Доход

## Задача

Исследовать, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок
