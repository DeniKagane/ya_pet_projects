# Защита персональных данных клиентов


## Данные

В наличии были данные клиентов:
- Пол
- Возраст
- Зарплаты
- Члены семьи
- Страховые выплаты

## Задача

Защитить данные, чтобы при преобразовании качество моделей машинного обучения не ухудшилось

## Используемые библиотеки
*pandas*  
*numpy*  
*scikit-learn*
